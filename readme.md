# Portfólio Pessoal - Kleberson Vasconcelos

Este projeto é um portfólio pessoal desenvolvido para apresentar o perfil profissional, habilidades, projetos e canais de contato de **Kleberson Vasconcelos**, com foco em uma interface moderna, responsiva e visualmente marcante.

O site foi construído com tecnologias front-end básicas, sem dependências externas de build, o que facilita a edição, execução e publicação.

## Visão geral

O portfólio inclui:

- seção inicial com apresentação profissional
- seção "Sobre Mim"
- cards de projetos
- seção de habilidades com ícones de tecnologias
- área de contato com links rápidos
- alternância entre tema claro/escuro com persistência no `localStorage`
- animação de entrada para os cards de habilidades usando `IntersectionObserver`

## Tecnologias utilizadas

- `HTML5`
- `CSS3`
- `JavaScript`
- `Google Fonts`

## Estrutura do projeto

```text
Projeto-Portfolio/
├── assets/
│   └── images/
├── css/
│   └── style.css
├── js/
│   └── main.js
```

## Como executar

Como este é um projeto estático, basta abrir o arquivo `Projeto-Portfolio/index.html` no navegador.

Se preferir, você também pode usar uma extensão como `Live Server` no VS Code para visualizar o site com recarregamento automático.

## Funcionalidades

- layout de página única
- navegação por âncoras no menu
- design com estética neon/dark
- cards com efeito hover
- modo escuro/claro com salvamento da preferência do usuário
- organização simples para manutenção e evolução do portfólio

## Pontos para evolução

- substituir os links `#` dos projetos por URLs reais
- adicionar imagens dos projetos que ainda não estão na estrutura atual
- corrigir textos com problema de codificação de caracteres
- melhorar a responsividade em telas menores, se necessário
- adicionar mais projetos reais e descrições detalhadas
